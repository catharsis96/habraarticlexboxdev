Разработка игр под Xbox на Windows 10

На ежегодной конференции Build 2016 компания Microsoft представила приложение Xbox Dev Mode, при помощи которого можно активировать режим разработчика. Благодаря этому независимые разработчики получат все необходимые инструменты для удобного создания и запуска на Xbox One собственных игр и приложений, а также дальнейшей загрузки их в магазин. Игровое сообщество получит множество новых игр и будет развиваться как никогда активно.

Давайте подробно рассмотрим процесс активации режима разработчика и некоторые особенности настройки управления, необходимой для корректной работы игр на Xbox One, на примере игрового движка Unity3D.

Режим разработчика

Xbox One поддерживает два режима работы: коммерческий и режим разработчика. В коммерческом режиме можно играть в игры и запускать приложения в качестве пользователя. В режиме разработчика можно разрабатывать программное обеспечение для консоли, но невозможно играть в коммерческие игры или запускать коммерческие приложения.

Вы можете активировать режим разработчика на любой коммерческой консоли Xbox One. Затем можно свободно переключаться между коммерческим режимом и режимом разработчика.

!

 Если вы являетесь участником программы предварительной версии, активировать режим разработчика будет невозможно. Сначала придется выйти из действующей программы с помощью информационной панели предварительной версии Xbox.

 

Активация режима разработчика на Xbox one

  1. Запустите консоль Xbox One.
  2. Выполните поиск и установите приложение Dev Mode Activation.

  3. Перейдите в меню Мои игры и приложения > Приложения.

  4. Откройте приложение Dev Mode Activation. Отобразится код, который в дальнейшем потребуется для подключения к Центру разработки.

  5. Войдите в Центр разработки https://developer.microsoft.com под своей учетной записью.
  6. Откройте страницу https://developer.microsoft.com/xboxactivate .
  7. Введите код активации, который отображается в приложении Dev Mode Activation на Xbox.

    8. !

    		 Количество активаций для каждой учетной записи ограничено. После активации режима разработчика Центр разработки сообщит, что вы использовали одну из активаций, связанных с вашей учетной записью.

     

  8. Выберите пункт Принять и активировать. Страница перезагрузится, а ваше устройство будет внесено в таблицу.

  9. После этого на консоли появится экран хода выполнения активации.
  10. После завершения процесса активации откройте приложение Dev Mode Activation и щелкните Переключиться и перезапустить, чтобы перейти в режим разработчика. Перезапуск консоли займет больше времени, чем обычно.

!

 В режиме разработчика невозможно запускать купленные игры и приложения, отсутствует доступ к сообществу, магазину, OneGuide и службе Xbox Live. Чтобы воспользоваться этими функциями, вернитесь в коммерческий режим.

 

Переключение между коммерческим режимом и режимом разработчика

Чтобы перейти в коммерческий режим, зайдите главную страницу разработки на Xbox One и щелкните Закрыть режим разработчика. После этого консоль перезапустится в коммерческом режиме.

Чтобы переключиться в режим разработчика, откройте на Xbox One приложение Dev Mode Activation. Выберите пункт Переключиться и перезапустить. После этого консоль перезапустится в режиме разработчика.

Настройки управления

Перед развертыванием игры на Xbox One нужно обеспечить их правильное взаимодействие.

Благодаря Universal Windows Platform (UWP), платформе для разработки универсальных приложений, разработчики могут создавать игры и приложения, которые, используя один и тот же код, запускаются на разных устройствах: ПК, смартфонах, Xbox и т.д. Поэтому для корректной работы на Xbox One игры, созданной при помощи игрового движка Unity3D, необходимо лишь провести простую настройку средств управления:

  1. Откройте проект в Unity перейдите в Edit > Project Settings > Input
  2. В окнеInspector можно будет настроить элементы управления. Можно отредактировать дефолтные значения или задать собственные.

В поле Size указано количество элементов управления. Задайте необходимое вам значение и перейдите к назначению кнопок.

В поле Name указывается любое удобное имя кнопки или стика, к которому потом нужно будет обращаться в коде.

Для настройки стика в поле Type выберите Joystick Axis, а в Axis нужную вам ось.

Для настройки кнопки в поле Type выберитеKey or Mouse Button, а в поле Positive Button напишите «joystick button x», где х – это номер кнопки.

Развертывание игр на Xbox

Теперь, когда все настроено, можно заливать игру на Xbox.

  1. Открываем наш проект в Unity и переходим в настройки билда:File > Build Settings и выбираем целевую платформуWindows Store. В поле SDK выбираемUniversal 10 и ставим чекбокс Unity C# Projects. Нажимаем Switch Platform и затем Build.

  2. После этого создается Visual Studio Solution, открываем его.
  3. Настроим параметры деплоя. Указываем Master, разрядность x64 и Remote Machine.
  4. На шаге выбора Remote Machine Visual Studio спросит IP-адрес Xbox. Если он находится в одной локальной сети с компьютером, то Xbox обнаружится автоматически.

  5. Нажимаем Remote Machine, инициируя деплой на Xbox.
  6. Готово! Игра загружена на Xbox, можно тестить.
